## Overview

Mirror of Truth is a local-first Android journaling and productivity app. It follows Clean Architecture layered pattern with MVVM in the presentation layer. All data lives on device; there are no remote integrations.

## System Diagram

```
PRESENTATION LAYER
  Screens (Composables) + ViewModels + MVI State
  home/ - journal/ - insights/ - settings/ - navigation/

DOMAIN LAYER
  Models (Task, Goal, JournalEntry)
  Repository Interfaces - UseCase placeholders

DATA LAYER
  Room DAOs - Entities - Mapper - RepositoryImpl
  MirrorDatabase (SQLite via Room)

DI (Hilt)

  Theme (Color, Type, Theme) - Utilities
```
